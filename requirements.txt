# ============================================================================
# Notas de Instalação:
# ============================================================================
#
# 1. Para Raspberry Pi, instale primeiro as dependências do sistema:
#    sudo apt-get update
#    sudo apt-get install -y python3-pip python3-opencv
#    sudo apt-get install -y libatlas-base-dev libhdf5-dev libhdf5-serial-dev
#    sudo apt-get install -y libjasper-dev libqtgui4 libqt4-test
#    sudo apt-get install -y i2c-tools python3-smbus
#
# 2. Para Picamera2 (Raspberry Pi 5 ou superior):
#    sudo apt-get install -y python3-picamera2
#
# 3. Habilite I2C e Camera na Raspberry Pi:
#    sudo raspi-config
#    -> Interface Options -> I2C -> Enable
#    -> Interface Options -> Camera -> Enable
#
# 4. Instale este requirements.txt:
#    pip3 install -r requirements.txt
#
# 5. Configure sua GROQ_API_KEY no arquivo config.json
#
